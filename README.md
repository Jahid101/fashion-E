## Description

[Project101](https://) starter repository. To run the app have to put in the environment files in the root directory(if needed).

## Installation

```bash
$ npm install
```


## Running the app

```bash

# watch mode
$ npm run dev

# development
$ npm run build


# production mode
$ npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
